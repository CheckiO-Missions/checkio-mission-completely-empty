You need to figure is this list completely empty. List is completely empty when all list inside of this list are empty.

Input: List of lists of lists ...

Output: True or False

Example: 

    completely_empty([]) == True
    completely_empty([1]) == False
    completely_empty([[]]) == True
    completely_empty([[],[]]) == True
    completely_empty([[[]]]) == True

Precondition: list
